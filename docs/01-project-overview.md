# AI Job Match Tracker

## Overview

AI Job Match Tracker is a full-stack web application that helps job seekers compare their resume against a job description using Artificial Intelligence.

The application provides:

- ATS Match Score
- Missing Skills Detection
- Resume Improvement Suggestions
- AI-generated Interview Questions
- Job Application Tracking

---

## Goal

Build a production-ready AI application demonstrating:

- React
- Node.js
- TypeScript
- Express
- PostgreSQL
- OpenAI API
- Docker
- GitHub Actions
- System Design
- Testing

---

## Target Users

- Students
- Software Engineers
- Career Changers
- Recruiters
