# AI Job Match Tracker

AI-powered full-stack application that helps job seekers compare their resume with job descriptions, calculate ATS match scores, identify missing skills, rewrite resume bullets, generate interview questions, and track job applications.

## Tech Stack

- React
- TypeScript
- Node.js
- Express
- PostgreSQL
- OpenAI API
- JWT Authentication

## Main Features

- Resume and job description comparison
- AI-powered match score
- Missing skills detection
- Resume bullet improvement
- Interview question generator
- Job application tracker
